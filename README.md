# 2026-campusroom-backend 🛡️

**Backend API** untuk Sistem Manajemen Peminjaman Ruangan Kampus.
Dibangun menggunakan **ASP.NET Core**.

## 🌟 Fitur Utama
* **Authentication & Authorization**: Sistem login aman menggunakan JWT (JSON Web Token).
* **Room Management**: CRUD (Create, Read, Update, Delete) data ruangan dengan status ketersediaan.
* **Booking System**: Logika peminjaman ruangan dengan validasi jadwal bentrok.
* **Role-Based Access**: Pemisahan hak akses antara **Admin** dan **Mahasiswa**.

## 🛠️ Teknologi yang Digunakan
* **Framework**: ASP.NET Core Web API (.NET 8.0)
* **Database**: SQL Server
* **ORM**: Entity Framework Core
* **Auth**: JWT Bearer Authentication
* **Documentation**: Swagger UI
